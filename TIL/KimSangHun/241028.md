# 요약: 아이디어 기획, 아키텍처 설계

# 주제: 치매 예방 관리

## 대상자: 치매 예방에 관심이 있는 7~80대(한글 아심)

## 서비스

### 게임

- 사물그리기
- 시물기억해서 그리기
- 가위바위보
- 숫자 순서대로 누르기
- 카드 짝 맞추기

### 통계

- 나이대
- 과거 비교

### 로그인

- 당사자
- 보호자

## 기획 결과

기획 해보았지만, 피드백이 좋지 못했다. 새로운 기획이 사실 필요한지 여부도 잘 생각하지 못했던 것 같다. 오픈소스에도 안 맞는 것 같다.

따라서 기존 프로젝트를 수행하는 것으로 바꾸었다. 기존 프로젝트도 할 수 있는 방법이 있기 때문에, 진행할 수 있을 것 같다.
