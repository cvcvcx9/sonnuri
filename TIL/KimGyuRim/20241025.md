# 중간발표 피드백

## Q&A

- 고려했으면 하는 부분 청각장애인 부분 토끼, 거북이 의미 잘 알까? 다른 오해를 일으킬 수 있지 않을까?
- 기술 구현 시간 많이 걸릴 것 같은데 사용자가 편하게 사용할 수 있을까?
  - 우리의 중점도 빠르게 구현하는 것
- 도메인 오픈소스와 프로젝트와 연관성?
  - 텍스트를 영상으로 바꿀 수 있는 기술 오픈소스로 활용하면 많은 곳에서 발전될 수 있다고 생각함
